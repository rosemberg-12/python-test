## Exercises

1. TreeLeafSum.py:
```text
Given the root of a binary tree, return the sum of all left leaves.

Example 1:

Given this tree below, the sum of all left leaves are 9 + 15 = 24

        3
       / \
      9  20
         / \ 
        15  7
        
Example 2:

Given this tree below, the sum of all left leaves are 2 + 3 = 5

        1      
       / \
      3   7     
     /   / \
    2   3   6
```
